# Bitvue Sample Video Files

This directory contains sample video files for testing all supported container and codec formats.

## Files

| File | Container | Video Codec | Description |
|------|-----------|-------------|-------------|
| `sample_h264.mp4` | MP4 | H.264/AVC | MP4 container with H.264 video |
| `sample_hevc.mp4` | MP4 | H.265/HEVC | MP4 container with HEVC video |
| `sample_vp9.mkv` | MKV | VP9 | Matroska container with VP9 video |
| `sample_vp9.webm` | WebM | VP9 | WebM container with VP9 video |
| `sample_vp9.ivf` | IVF | VP9 | IVF raw container with VP9 frames |
| `sample_av1.ivf` | IVF | AV1 | IVF raw container with AV1 frames |
| `sample_h264.h264` | Annex B | H.264/AVC | Raw H.264 byte stream |
| `sample_hevc.hevc` | Annex B | H.265/HEVC | Raw HEVC byte stream |

## Specifications

All samples are:
- **Duration**: 3 seconds
- **Resolution**: 640x360
- **Frame Rate**: 30 fps
- **Content**: Synthetic test pattern (testsrc2)
- **License**: Public domain (generated by FFmpeg lavfi)

## Usage

```bash
# Open in Bitvue
bitvue analyze samples/sample_h264.mp4

# Or use the Tauri app
# File → Open → select any sample file
```

## Generation

These samples were generated with FFmpeg:

```bash
# H.264 MP4
ffmpeg -f lavfi -i testsrc2=duration=3:size=640x360:rate=30 \
  -c:v libx264 -preset ultrafast -crf 28 -pix_fmt yuv420p sample_h264.mp4

# HEVC MP4
ffmpeg -f lavfi -i testsrc2=duration=3:size=640x360:rate=30 \
  -c:v libx265 -preset ultrafast -crf 28 -pix_fmt yuv420p sample_hevc.mp4

# VP9 MKV/WebM/IVF
ffmpeg -f lavfi -i testsrc2=duration=3:size=640x360:rate=30 \
  -c:v libvpx-vp9 -b:v 500k -deadline realtime -pix_fmt yuv420p sample_vp9.mkv

# Raw H.264 (Annex B)
ffmpeg -f lavfi -i testsrc2=duration=3:size=640x360:rate=30 \
  -c:v libx264 -preset ultrafast -crf 28 -pix_fmt yuv420p -f h264 sample_h264.h264

# Raw HEVC (Annex B)
ffmpeg -f lavfi -i testsrc2=duration=3:size=640x360:rate=30 \
  -c:v libx265 -preset ultrafast -crf 28 -pix_fmt yuv420p -f hevc sample_hevc.hevc
```
